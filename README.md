# Online News Newspaper - Course Project

A modern web application simulating an online newspaper with personalized news feeds, subscriptions, and content management.

## Tech Stack

- **Backend**: Python 3.9+ with Flask
- **Frontend**: React with JavaScript
- **Database**: MySQL 8.0+
- **Design Patterns**: GoF patterns (Strategy, Factory, Observer, Singleton, Repository)

## Features

1. **News Feed**: Latest publications, categorized sections, search functionality
2. **Content Personalization**: Favorite topics, recommendation system
3. **User Profile**: Comments, ratings, saved articles, subscriptions
4. **Subscription System**: Free (with ads), Paid (ad-free + exclusive), Special packages
5. **Notifications**: Breaking news, daily digest, author alerts
6. **Admin Panel**: Content management, user administration

## Setup Instructions

### Prerequisites
- Python 3.9+
- Node.js 16+
- MySQL 8.0+
- npm or yarn
